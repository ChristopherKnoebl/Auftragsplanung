# Auftragsplanung
Eine kleine Anwendung mit der Aufträge und Lagerbestände organisiert werden können.

## Idee
Es ist teilweise ärgerlich, wenn Fertigungsaufträge freigegebenw werden, ohne dass für diese ausreichend Material im Lager vorrätig ist. Diese Anwendung überprüft den Lagerbestand bevor ein Auftrag freigegeben wird und liefert eine Rückmeldung.
